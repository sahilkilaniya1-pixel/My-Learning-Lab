fieldset & legend: These group complex forms visually and logically (improved for accessibility). label[for] and input[id]: The value of the for attribute must match the id of the input. This means that whenever the user clicks on the label, the input will automatically be focused. required, minlength: These provide Native HTML Validation without JavaScript. The browser will not allow forms to be submitted unless these rules are followed. Controls: Play, pause, and volume buttons are provided. Poster: The image that will be seen before the video is loaded. Multiple tags: If the browser does not support one format (mp4), it will try another (webm).
